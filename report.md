# 📊 Multi-Agent System Report

## 1. System Overview

This system uses a multi-agent architecture to automate inventory, quoting, and sales processes.

## 2. Agent Roles

* Orchestrator: Controls workflow
* Inventory Agent: Checks stock and supplier data
* Quote Agent: Generates pricing strategies
* Sales Agent: Handles transactions
* Finance Agent: Tracks financial data

## 3. Workflow

Customer request → Inventory check → Quote generation → Sales processing → Finance update

## 4. Evaluation Results

* Multiple successful transactions recorded
* Cash balance updated correctly
* Some requests failed due to insufficient stock

## 5. Strengths

* Modular design
* Scalable architecture
* Efficient automation

## 6. Improvements

* Add dynamic pricing model
* Improve supplier forecasting
